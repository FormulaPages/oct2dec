# dec2bin

[![Build Status](https://travis-ci.org/FormulaPages/oct2dec.svg?branch=master)](https://travis-ci.org/FormulaPages/oct2dec)


## Install

```sh
npm install formula-oct2dec
```

### Usage

```js
var oct2dec = require('formula-oct2dec');
oct2dec(54) // => 44
oct2dec(7777777533) // => -165
```
